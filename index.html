8052 Electronic Organ

This was a simple Embedded C project. The 8052-based microcontroller was interfaced with a 4x4 keypad, an LCD and a speaker to produce 16 different frequencies. The LCD would display the current note and the frequency. The objective of this project was to have a thorough understanding of interrupt based calculations and the interfacing of the microcontroller with LCD and keyboard.

Mini C Compiler

This project was aimed at implementing a basic C compiler using Flex & Bison. The main goals were to identify syntactical integrity for some key elements of C programming - namely iterative and conditional structures, printing statements, and basic C structure.

Café Management System

Built using NetBeans and Java, integrated with Dropbox API and documented with UML, this project resulted in development of a software which a particular cafe could use for its day-to-day activities, like sale/purchase of goods, creation of bill, profit/loss, management of expired products, etc.

Used Vehicle Dealership System

Built using C# on Microsoft Visual Studio with MySQL, this software provides four different types of view levels, depending on the user category. A used vehicle dealership store could use it for sale/purchase of vehicles, creation of bills, profit/loss, insurance management, partial payment management, etc.
